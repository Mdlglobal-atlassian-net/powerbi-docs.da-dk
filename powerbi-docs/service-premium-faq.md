# <a name="power-bi-premium-faq"></a>Ofte stillede spørgsmål om Power BI Premium
Gennemse en liste med ofte stillede spørgsmål og svar om tilbuddene i Power BI Premium.

* Hvis du har andre spørgsmål, kan du [prøve at spørge Power BI-community'et](http://community.powerbi.com/).
* Har du stadig problemer? Besøg [supportsiden for Power BI](https://powerbi.microsoft.com/support/).
* Eller klik på **Kontakt mig**-ikonet ![](media/service-premium-faq/power-bi-contact-me-icon.png) i det nederste højre hjørne i dette browser-vindue.

## <a name="power-bi-premium"></a>Power BI Premium
**Hvad er Power BI Premium?**  
Power BI Premium er et nyt kapacitet-baserede tilbud, der omfatter:

* Fleksibilitet til at publicere rapporter bredt på tværs af en virksomhed, uden at modtagerne skal have en licens hver for sig pr. bruger.
* Større skala og ydeevne fra dedikeret kapacitet i Power BI-tjenesten.
* Muligheden for at have BI-aktiver i det lokale miljø med Power BI Report Server.
* En API-brugergrænseflade, en konsistent række funktioner og adgang til de nyeste funktioner for integrerede analytics.

**Hvad gør Power BI Premium? Hvordan fungerer det?**  
Power BI Premium består af kapacitet i Power BI-tjenesten, der udelukkende er allokeret til den enkelte organisation, og som understøttes af dedikeret hardware, der administreres fuldt ud af Microsoft. Organisationer kan vælge at anvende deres dedikerede kapacitet bredt eller tildele den til arbejdsområder, afhængigt af antallet af brugere, arbejdsbelastningsbehov eller andre faktorer samt skalere op eller ned, i takt med at kravene ændres.

**Hvordan adskiller Power BI Pro sig fra Power BI Premium? Hvorfor er Power BI Premium nødvendig?**  
Power BI Premium er en kapacitet-baseret licens, mens Power BI Pro er en brugerbaseret licens.

Power BI Premium består af kapacitet i Power BI-tjenesten, der udelukkende er allokeret til den enkelte organisation, og som understøttes af dedikeret hardware, der administreres fuldt ud af Microsoft, hvorfor begrænsninger for datasætstørrelse og opdateringshastigheder fjernes, og isolation understøttes takket været den dedikerede hardware. Power BI Premium giver også fleksibilitet til at publicere rapporter bredt på tværs af en virksomhed, uden at modtagerne skal have en licens hver for sig pr. bruger.

Power BI Pro er til de brugere, der publicerer rapporter, deler dashboards, samarbejder med kolleger i arbejdsområder og er engageret i andre relaterede aktiviteter og f.eks.:

* Redigerer og gemmer brugerdefinerede visninger.
* Oprette personlige dashboards (fastgøres til nye dashboard).
* Analyserer data i Excel eller Power BI Desktop.
* Deler med understøttelse af Excel Web App.
* Deler dashboards og samarbejder med Office 365 Grupper.
* Integrerer indhold med Microsoft Teams.

**Skal jeg have Power BI Pro for at kunne bruge Power BI Premium?**  
Ja. Power BI Pro er nødvendigt for at kunne publicere rapporter, dele dashboards, samarbejde med kolleger i arbejdsområder og foretage andre relaterede aktiviteter.

**Hvor meget koster Power BI Premium? Hvor SKU'er bliver tilgængelige?**  
Du kan købe Power BI Premium baseret på antallet af virtuelle kerner. Du kan se priser på [siden med Power BI-priser](https://powerbi.microsoft.com/pricing/). Du kan få flere oplysninger om noder og v-kerner i [Microsoft Power BI Premium-hvidbogen](https://aka.ms/pbipremiumwhitepaper). Du kan også bruge denne [beregning](https://powerbi.microsoft.com/calculator/) til at få et estimat over, hvor meget Premium-kapacitet du kan få brug for.

**Hvad menes der med "kapacitet"?**  
"Kapacitet" refererer til en navngivet kapacitet, der klargøres af en administrator via portalen til administration af kapacitet til Power BI Premium. En kapacitet er en gruppering af en eller flere noder. Du kan få flere oplysninger i [Microsoft Power BI Premium-hvidbogen](https://aka.ms/pbipremiumwhitepaper).

**Hvordan faktureres Power BI Premium?**  
Power BI Premium faktureres månedligt med en årlig binding.

**Hvordan køber jeg Power BI Premium?**  
Power BI Premium er tilgængelig som et tilføjelsesprogram til Power BI Pro i O365 Admin Center. Du kan finde flere oplysninger under [Sådan køber du Power BI Premium](service-admin-premium-purchase.md). Du kan også kontakte din Microsoft-repræsentant for at få flere oplysninger.

**Hvordan kan jeg vide, hvor meget Power BI Premium-kapacitet jeg skal købe?**  
Du kan bruge denne [beregning](https://powerbi.microsoft.com/calculator/) til at få et estimat over, hvor meget Power BI Premium-kapacitet du kan få brug for.

**Hvornår er Power BI Premium tilgængelig? Bliver det tilgængeligt samtidigt på tværs af alle lande?**  
Power BI Premium bliver offentligt tilgængeligt i slutningen af andet kvartal i 2017 i alle lande, der understøttes i øjeblikket af Power BI-tjenesten.

**Opfylder Power BI Premium kravene til overholdelse af certificeringer og lovgivningsmæssige krav på tidspunktet for udgivelsen? Hvad med nationale clouds?**  
Power BI Premium bliver offentligt tilgængeligt i slutningen af andet kvartal i 2017 i alle lande, der understøttes i øjeblikket af Power BI-tjenesten. Nationale clouds – specielt Tyskland, Kina og den amerikanske regeringer – planlægges at være tilgængelige i 3. kvartal i 2017. Standarder for Power BI-certificering og overholdelse af lovmæssige krav gælder for Power BI Premium på det tidspunktet, hvor det bliver tilgængelige generelt.

**Fås Power BI Premium sammen med Office 365 E5?**  
Power BI Premium fås som et tilføjelsesprogram til Power BI Pro. Office 365 E5 indeholder Power BI Pro. E5-kunder kan købe Premium som et tilføjelsesprogram til deres eksisterende Pro-licenser.

**Hvad er Power BI Report Server? Betyder det, at Power BI bliver tilgængeligt i det lokale miljø?**  
Power BI Report Server er tilgængelig som en del af Power BI Premium og er kompatibel med SQL Server Reporting Services-rapporter, så installation og distribution af interaktive rapporter i Power BI og sideinddelte rapporter i det lokale miljø bliver muligt inden for grænserne af en organisations firewall. Organisationer kan vælge Power BI i clouden eller vælge at beholde rapporter i det lokale miljø med Power BI Report Server og flytte til clouden i eget tempo.

Power BI-funktioner som dashboards i realtid, naturlige sprogforespørgsler og andre funktioner i Power BI-tjenesten er ikke tilgængelige som en del af Power BI Report Server. 

Yderligere oplysninger om Power BI Report Server finder du i [Kom i gang med Power BI Report Server](report-server/get-started.md).

**Er Power BI Report Server den funktion, som Microsoft har gjort tilgængelig som Power BI-rapporter til SSRS technical preview? Hvorfor er det nu en del af Power BI, når det virkede som om, at det var en SQL-funktion?**  
Power BI Report Server er tilgængelig som en del af Power BI Premium og er kompatibel med SQL Server Reporting Services-rapporter, så installation og distribution af interaktive rapporter i Power BI og sideinddelte rapporter i det lokale miljø bliver muligt inden for grænserne af en organisations firewall.

Bekendtgørelsen om Power BI Premium er første gang, vi har afsløret, hvordan denne funktionalitet lanceres.

Yderligere oplysninger om Power BI Report Server finder du i [Kom i gang med Power BI Report Server](report-server/get-started.md).

**Hvad er Power BI-apps?**  
Vi udvikler indholdspakker til Power BI-apps for at forbedre, hvordan brugerne søger efter og udforsker indsigt på koncernniveau. Power BI-apps, der fås i dag, tilbyder en forenklet måde at udrulle dashboards og rapporter til bestemte brugere, grupper eller en hel organisation. Forretningsbrugere kan let installere disse apps og ubesværet navigere i dem, da indhold centraliseres ét sted og opdateres automatisk. 

Du kan finde flere oplysninger om apps under [Hvad er apps i Power BI?](service-install-use-apps.md).

**Foretages der ændringer i Power BI Pro, Power BI Free Power BI Desktop og andet som en del af udrulningen af Power BI Premium?**  
I dag forenkler vi også forskellen mellem Power BI Pro og den gratis tjeneste. Vi har modtaget feedback, at eftersom den gratis tjeneste er beregnet til privat brug og Power BI Pro muliggør samarbejde, har de funktionelle forskelle mellem dem skabt forvirring for brugerne. Fremover vil vi forbedre den gratis tjeneste for at have den samme funktionalitet som Power BI Pro, men vil begrænse delings og samarbejdsfunktioner til Power BI Pro-brugere alene.

Brugere af den gratis Power BI-tjeneste kan drage fordel af adgang til alle datakilder, større lagerplads for arbejdsområder samt højere opdaterings- og streaminghastigheder. Ændringerne trådte i kraft den 1. juni 2017, og du kan læse mere under [Tilmelding via selvbetjening til Power BI](service-self-service-signup-for-power-bi.md). Power BI Desktop bliver ved med at være gratis.

Du kan finde oplysninger om den udvidede Pro-prøveversion under [Aktivering af den udvidede Pro-prøveversion](service-extended-pro-trial.md).

## <a name="power-bi-free"></a>Power BI (gratis)
**Hvilke funktioner føjes til den gratis tjeneste?**  
Fra den 1. juni 2017 har den gratis tjeneste funktioner, der svarer til Power BI Pro, undtagen deling, distribution på virksomhedsniveau og samarbejde. Dette omfatter denne samme 10 GB-begrænsning for arbejdsområder, op til 8 daglige planlagte opdateringer for datasæt og en streamingdatahastighed på maks. 1 mio./time. Der er også adgang til alle datakilder, herunder dem, der er tilgængelige via datagatewayen i det lokale miljø.

**Hvilke Power BI Pro-funktioner er ikke tilgængelige i den gratis tjeneste?**  
Deling af peer to peer-dashboard, apparbejdsområder (tidligere kaldet gruppearbejdsområder), Analysér i Excel og Power BI-apps (medmindre der er Premium-kapacitet) er funktioner, der kun fås i Power BI Pro.

**Hvornår træder disse ændringer i kraft?**  
Ændringer af den gratis tjeneste trådte i kraft den 1. juni 2017.

**Hvordan får de brugere af den gratis tjeneste, der er berørt af ændringerne, besked om disse ændringer?**  
Aktive brugere af den gratis tjeneste, der er berørt af ændringerne den 1. juni 2017, modtager en mail fra Microsoft samt meddelelser, når de logger på Power BI-tjenesten fra den 3. maj 2017.

Eksisterende brugere af den gratis tjeneste, som har været aktive inden for det sidste år, kan drage nytte af den gratis 12-måneders udvidet prøveversion af Power BI Pro. Tilbuddet giver dig de fulde funktioner i Power BI Pro for at sikre, at brugerne har tilstrækkelig med tid til at justere, hvordan du bruger tjenesten. Du kan finde oplysninger om den udvidede Pro-prøveversion under [Aktivering af den udvidede Pro-prøveversion](service-extended-pro-trial.md).

**Mister brugerne adgang til deres indhold (dashboards, rapporter) på dette tidspunkt?**  
Nej. De mister ikke adgang til indhold, de har overført til Power BI-tjenesten. Indhold, der deles med andre, er muligvis ikke længere tilgængelige for modtagerne den 1. juni 2017. På dette tidspunkt har de muligvis heller ikke længere adgang til indhold, som andre har delt med dem.

Eksisterende brugere af den gratis tjeneste, som har været aktive inden for det sidste år, kan drage nytte af den gratis 12-måneders udvidet prøveversion af Power BI Pro. Tilbuddet giver dig de fulde funktioner i Power BI Pro for at sikre, at brugerne har tilstrækkelig med tid til at justere, hvordan du bruger tjenesten. Du kan finde oplysninger om den udvidede Pro-prøveversion under [Aktivering af den udvidede Pro-prøveversion](service-extended-pro-trial.md).

**Hvordan ved man som bruger, om man er kvalificeret til den udvidede Power BI Pro-prøveversion?**  
Se [vilkår og betingelser](https://aka.ms/power-bi-trial) for den udvidede Pro-prøveversion.

Du kan finde oplysninger om den udvidede Pro-prøveversion under [Aktivering af den udvidede Pro-prøveversion](service-extended-pro-trial.md).

**Hvad skal man som bruger gøre, hvis man ikke er kvalificeret til den udvidede prøveversion af Power BI Pro?**  
Tilmeld dig en 60-dages standardprøveversion af Power BI Pro [her](https://app.powerbi.com/signupredirect?pbi_source=web).

**Ændrer lanceringen af Power BI Premium funktionerne i den gratis tjeneste?**  
Brugerne af den gratis tjeneste med adgang til dedikeret kapacitet i Power BI Premium får mulighed for at modtage indhold, der er distribueret til dem af Power BI Pro-brugere via publicerede apps.

## <a name="power-bi-pro"></a>Power BI Pro
**Hvad ændres med Power BI Pro?**  
Der foretages ikke ændringer af Power BI Pro.

**Hvad med Office – er Power BI Pro stadig omfattet af Office 365 E5?**  
Ja. Power BI Pro vil fortsat være omfattet af Office 365 E5.

**Jeg kan vende tilbage til mine Power BI Pro-licenser, hvis jeg vil skifte til Power BI Premium?**  
Nej. Power BI Premium er et tilføjelsesprogram til Power BI Pro. Premium kræver derfor Pro-licenser.

**Hvorfor er Power BI Pro nødvendigt? Kan jeg ikke nøjes med at købe Power BI Premium?**  
Power BI Premium er en kapacitet-baseret licens, mens Power BI Pro er en bruger-baseret licens, der er beregnet til hurtige og brugervenlige selvbetjeningsanalyser for de brugere, der har behov for samarbejde, deling af dashboard, ad hoc-analyser og rapportudgivelser og f.eks muligheden for at:

* Redigere og gemme brugerdefinerede visninger.
* Oprette personlige dashboards (fastgøres til nye dashboard).
* Analysere data i Excel eller Power BI Desktop.
* Dele og få vist Excel-projektmapper med Excel Web App-understøttelse.
* Dele dashboards og samarbejde med Office 365 Grupper.
* Integrere indhold med Microsoft Teams.

## <a name="power-bi-embedded"></a>Power BI Embedded
**Hvad er blevet ændret med Power BI Embedded?**  
Som en del af Power BI Premium konvergerer vi Power BI Embedded med Power BI-tjenesten for at levere en API-overflade, et konsistent sæt funktioner og adgang til de nyeste funktioner. Fremover opfordrer ved dem, der er interesseret i at integrere Power BI i deres apps, til at starte med Power BI Desktop og flytte til installation med Power BI Premium.

**Hvad er fordelene ved Power BI Premium i forbindelse med integration af analyser i apps?**  
Power BI Premium er et kapacitet-baseret tilbud, der kører på dedikeret hardware, og som administreres fuld ud af Microsoft som en del af Power BI-tjenesten. I forhold til livscyklussen for appudvikling tilbyder Power BI Premium nye fordele i form af fleksible og forudsigelige omkostninger, forbedret indholdsadministration for udviklere og fuld tilgængelighed til funktioner i Power BI for integrerede scenarier.

**Skal jeg overføre min eksisterende app, der er baseret på Power BI Embedded?**  
Power BI Embedded vil fortsat være tilgængelig for eksisterende apps pr. følgende tidsplan:

| Licensaftale | Tilgængelighed af Power BI Embedded |
| --- | --- |
| Enterprise-aftale |Indtil udløb af eksisterende aftale |
| Direkte og CSP |Et år, fra Power BI Premium bliver offentligt tilgængeligt |

Ja, du bør begynde at overveje at overføre din integrerede analytics-løsning til Power BI Premium. Du kan få flere oplysninger under [Sådan overfører du indhold fra samlinger i arbejdsområdet i Power BI Embedded til Power BI](developer/migrate-from-powerbi-embedded.md)

**Hvordan overfører jeg min eksisterende app, der er baseret på Power BI Embedded?**  
Se [dokumentationen til overførsel](developer/migrate-from-powerbi-embedded.md) for at få en vejledning i overførslen af apps.

**Går Microsoft tilbage til sit tilsagn om Power BI Embedded Free?**  
Under den forrige model tilbød vi et begrænset antal session til udvikling og test. I det nye scenarie har kunderne fortsat mulighed for at begynde at udvikle og teste gratis på deres måde for at være klar til produktion. Kunder kan drage fordel af en 2-måneders Power BI Pro-prøveversion til at komme i gang med udvikling og test. Power BI Desktop samt værktøjet til dataudforskning og rapportoprettelse vil fortsat være tilgængelig gratis.

**Er Power BI Embedded stadigvæk en Azure-tjeneste? Hvor kan jeg købe, klargøre og administrere Power BI Embedded?**  
Power BI Embedded konvergeres med Power BI-tjenesten og købes, klargøres og administreres derfor som en sådan tjeneste fremover. Du kan finde flere oplysninger under [Integration med Power BI](developer/embedding.md).

## <a name="power-bi-report-server"></a>Power BI Report Server
**Hvorfor er Power BI Report server inkluderet i Power BI Premium i stedet for at blive solgt som et separat produkt?**  
Power BI Premium gør det muligt at have BI-aktiver i det lokale miljø sammen med Power BI Report Server. Power BI Report Server er en server i det lokale miljø-server, der muliggør installation og distribution af interaktive rapporter i Power BI og sideinddelte rapporter inden for organisationens firewalls grænser. Med Power BI Premium kan det samme antal virtuelle kerner, som en organisation klargør i clouden, installeres i det lokale miljø via Power BI Report Server uden at skulle opdele kapaciteten. Organisationer kan vælge Power BI i clouden eller vælge at beholde rapporter i det lokale miljø med Power BI Report Server og flytte til clouden i eget tempo.

Du kan finde flere oplysninger i [Kom i gang med Power BI Report Server](report-server/get-started.md).

**Hvordan kan kunder få adgang til Power BI Report Server?**  
Power BI Report Server er tilgængelig via Power BI Premium eller som en fordel for kunder med SQL Server-EE med SA. Kontakt din Microsoft-repræsentant for at få flere oplysninger.

**Hvorfor er Power BI Report Server ikke tilgængelig som fordel for SQL Server Standard- eller Enterprise Edition-kunder?**  
Power BI Report Server er tilgængelig for SQL Server-kunder med EE- og SA-licenser eller til køb via Power BI Premium. Power BI er en SaaS-tjeneste, og Power BI Report Server følger den samme abonnementsbaserede leveringsmodel.
